# Mindloom

> Weave your thoughts into insight.

## About

Mindloom is an AI-assisted journaling platform designed to help users reflect on their daily experiences, emotions, achievements, and challenges. Users can write journals manually or use guided AI prompts that ask reflective questions and generate meaningful journal entries.

The goal of Mindloom is not only to store memories but to help users understand patterns in their emotions, build self-awareness, and track personal growth over time.

This project is being built as a full-stack MERN application with AI integration to learn backend development, database management, authentication, APIs, and production-level project structure.
